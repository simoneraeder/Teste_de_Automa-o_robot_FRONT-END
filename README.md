# Teste de Automação Front-End com Robot Framework

## 📌 Sobre o Projeto

Este projeto tem como objetivo demonstrar a automação de testes de uma aplicação **Front-End** utilizando **Robot Framework** em conjunto com **Selenium Library**.

A automação executa testes simulando ações de um usuário em um navegador, como:

* Acessar páginas
* Preencher formulários
* Clicar em botões
* Validar mensagens e elementos da interface

Esse tipo de automação é utilizado em processos de **Quality Assurance (QA)** para garantir que o sistema funcione corretamente após alterações no código.

## ⚙️ Pré-requisitos

Antes de executar o projeto é necessário ter instalado:

* Python 3.x
* Google Chrome ou outro navegador
* WebDriver correspondente ao navegador

---

## 📥 Instalação

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/Teste_de_Automacao_robot_FRONT-END.git
```

2. Acesse a pasta do projeto:

```
cd Teste_de_Automacao_robot_FRONT-END
```

3. Instale as dependências:

```
pip install -r requirements.txt
```

Ou manualmente:

```
pip install robotframework
pip install robotframework-seleniumlibrary
```

---

## ▶️ Executando os Testes

Para executar todos os testes do projeto utilize:

```
robot tests/
```

Após a execução, serão gerados arquivos de relatório.

---

## 📊 Relatórios Gerados

Depois da execução dos testes o Robot Framework gera automaticamente:

* **report.html** → resumo dos testes executados
* **log.html** → detalhes da execução
* **output.xml** → arquivo de resultado utilizado por ferramentas de integração
